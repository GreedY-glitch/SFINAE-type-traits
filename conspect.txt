Программа может отслеживать и изменять собственную структуру и поведение во время исполнения. Парадигма : рефлексия. 
Данная парадигма является ответвлением в метапрограммировании.

template <class B, class T = void> struct enable_if;

B - значение, определяющее наличие результирующего типа
T - тип для создания экзепляра, если B = true

!!!!!!!!!!!!!! IMPORTANT !!!!!!!!!!!!!!!!!!
Если B имеет значение true, enable_if<B, T> имеет вложенное определение типа с именем "type", которое является синонимом для T.

Если B имеет значение false, enable_if<B, T> не имеет вложенного определения типа с именем "type".
